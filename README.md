# NoirVoider Homebrew Tap

Homebrew tap for NoirVoider's macOS applications.

## Usage

```bash
brew tap NoirVoider/tap
brew install --cask worklog
```

Or directly:

```bash
brew install NoirVoider/tap/worklog
```

## Available Casks

| Cask | Description |
|------|-------------|
| `worklog` | A native-feeling macOS Markdown worklog app |
